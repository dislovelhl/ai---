# Workflows Overview

本文档定义了 AI 导航平台开发的标准工作流程。

## Available Workflows

| Workflow             | Description  | Use Case                     |
| -------------------- | ------------ | ---------------------------- |
| `new-feature`        | 新功能开发   | 添加新功能或模块             |
| `add-tool`           | 添加 AI 工具 | 手动添加新的 AI 工具到数据库 |
| `content-pipeline`   | 内容管线     | 处理自动爬取的内容           |
| `api-endpoint`       | API 开发     | 创建新的 API 端点            |
| `database-migration` | 数据库迁移   | 修改数据库结构               |
| `bug-fix`            | Bug 修复     | 修复已知问题                 |
| `document`           | 文档维护     | 维护和创建项目文档           |
| `performance`        | 性能优化     | 优化系统性能                 |
| `seo-update`         | SEO 更新     | 优化搜索引擎排名             |

## Quick Reference

```
/workflow new-feature     # 开始新功能开发
/workflow add-tool        # 添加新工具
/workflow bug-fix         # 修复Bug
```
